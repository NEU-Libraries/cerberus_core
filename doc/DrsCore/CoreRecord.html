    module DrsCore::CoreRecord

    This module implements the basic functionality for a core record in any head based off the DRS Architecture. Core records are fedora objects that hold metadata related to any number of content objects (defined in DrsCore::ContentObject) attached to them via the standard isPartOf relationship. Must be subclassed off an object extending ActiveFedora::Base

def content_objects
  a = self.content_classes

  a.inject! { |base, str| base + "OR \"#{str}\""}
  models = "active_fedora_model_ssi:(#{a})"
  belongs_to_this = "is_part_of_ssim:\"info:fedora/#{self.pid}\""

  as = ActiveFedora::SolrService
  query_result = as.query("#{models} AND #{belongs_to_this}", rows: 999)
  assigned_lookup(query_result)
end
